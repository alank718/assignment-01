# Assignment 01

## Problem 1: Cyclic Substring Maximum Sum
Given a string of lowercase alphabets, find the maximum sum of values of a cyclic substring such that all characters are unique.

**Approach:**  
Used a Sliding Window technique with a Set to maintain unique characters and efficiently compute the maximum sum.

---

## Problem 2: Array Transformation Cost Minimization
Given an array and a value K, determine the minimum number of operations required to make all elements equal by adding or subtracting K.

**Approach:**  
First checked feasibility using modulo condition. Then normalized the array and used the Median to minimize total operations.
